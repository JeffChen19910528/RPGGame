## 故事背景

三年前，魔王**夜陌魯斯**率領魔族大軍入侵，勇者在最後的戰役中身受重傷，從此昏迷不醒。

三年後，你睜開眼睛。

村裡的老祭司告訴你：**魔王的軍隊三天後就會到達這個村子。**

你是他們最後的希望——但你在昏迷的這三年裡，世界已然改變，你的心也在即將到來的旅程中，被迫面臨一次次的抉擇。

**這不只是一場打倒魔王的戰爭，而是一趟關於憤怒、選擇與自我的旅程。**

## 遊戲教學

### 主選單

遊戲啟動後，先選擇**語言**，再進入主選單：

```
  [1] 新遊戲
  [2] 讀取存檔
  [3] 遊玩說明

  [5] 執行程式測試
```

### 角色創建

#### 1. 輸入名字

輸入你的角色名稱（直接按 Enter 使用預設名）。

#### 2. 選擇職業（共 5 種）

| 職業 | HP | ATK | DEF | MP | 特色 |
|------|-----|-----|-----|-----|------|
| **戰士** | +30 | — | +5 | — | 高血量防禦，怒氣累積快 |
| **法師** | — | +3 | — | +30 | 高 MP，技能傷害強 |
| **刺客** | −10 | +8 | — | — | 極高攻擊，高風險打法 |
| **聖騎士** | +40 | −2 | +8 | — | 超高防禦，強力治癒與控制 |
| **遊俠** | — | +5 | — | +20 | 毒攻擊與精準射擊，均衡輸出 |

> 職業不僅影響數值，也決定你的專屬技能組、故事台詞與隱藏結局條件。

### 戰鬥系統

戰鬥採回合制，玩家先行。每回合選擇行動：

```
  [1] 普通攻擊
  [2] 使用技能
  [3] 防禦姿態（受到傷害減半）
  [4] 查看狀態
```

#### 傷害計算

```
最終傷害 = 基礎攻擊 × 技能倍率 × 隨機係數（±12%）− 敵人防禦
```

狂暴狀態下攻擊傷害額外 × 1.5，但有 **20% 機率反噬**（扣自身 HP）。

#### 狀態效果

| 狀態 | 效果 |
|------|------|
| **燃燒／中毒** | 每回合扣除約 9 點 HP，持續 3 回合 |
| **暈眩** | 跳過下一回合行動 |
| **防禦中** | 受傷減半，持續至下一次行動 |

### 憤怒／狂暴系統

怒氣值是本作的核心機制：

```
怒 ████████░░  80/100
```

- **累積方式**：普通攻擊 +15、受到傷害 +12、防禦時 +5
- **觸發條件**：怒氣達到 100 自動進入**狂暴狀態**（4 回合）
- **狂暴效果**：所有攻擊傷害 ×1.5，但 20% 機率反噬自身

> 狂暴狀態強大但並非毫無代價。如何運用憤怒，正是這段旅程的核心課題。

### 角色成長

擊敗敵人獲得 EXP，達標後升級：

| 升級效果 | 數值 |
|----------|------|
| 最大 HP | +20 |
| 最大 MP | +10 |
| 攻擊力 | +3 |
| 防禦力 | +2 |

達到 **3 級**後解鎖各職業的兩個進階技能。

## 幽默支線劇情

每章各藏有一條幽默支線，在緊張的冒險中提供喘息空間，同時帶來實際的資源獎勵。

遭遇什麼、怎麼應對——就留給你自己去發現吧。

## 結局一覽

共 **9 種結局**，由腐化值、道德選擇、職業與狂暴次數共同決定，其中包含數個需要特定條件才能觸發的隱藏結局。

你的每一個選擇都在累積，結局會說實話。

## 專案架構

```
RPGGame/
├── start.bat           # Windows 一鍵啟動
├── start.sh            # macOS/Linux 一鍵啟動
├── Program.cs          # 入口點（支援 --test 參數）
├── GameManager.cs      # 主遊戲迴圈、章節流程、存讀檔
├── Player.cs           # 玩家屬性、升級、技能、狂暴邏輯
├── Enemy.cs            # 敵人資料（13 種）、狀態效果、隨機遭遇池
├── BattleSystem.cs     # 戰鬥引擎：回合流程、行動解析
├── SkillSystem.cs      # 職業專屬技能定義與傷害計算
├── StoryManager.cs     # 劇情文本、幽默支線、分支選擇、9 種結局判定
├── GameTests.cs        # 自動化測試套件（134 項測試）
├── Localization.cs     # 多語系字串（ZH / EN / JA）
├── AnimationSystem.cs  # ASCII 動畫渲染
├── AsciiArt.cs         # ASCII 圖像資源
└── Utils.cs            # UI 工具：打字動畫、選單、進度條
```
