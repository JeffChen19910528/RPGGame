# RPGGame — 黑暗勇者傳說

一款以 C# / .NET 8 打造的純文字回合制 RPG，內含完整三章劇情、多重結局、ASCII 動畫戰鬥與道德選擇系統。

## 專案說明

| 項目 | 內容 |
|------|------|
| 語言 | C# |
| 執行環境 | .NET 8.0 |
| 相依套件 | 無（僅使用標準函式庫） |
| 存檔格式 | JSON |
| 平台 | Windows / macOS / Linux（終端機） |

### 核心特色

- **完整三章故事**，含分支對話與道德選擇，影響最終結局
- **三種角色職業**：戰士、法師、刺客，各有不同屬性加成
- **8 種技能**：4 種初始 + 4 種 3 級解鎖
- **狂暴系統**：怒氣累積至 100 觸發強化狀態，但帶有反噬風險
- **墮落值追蹤**：影響通關結局（善、惡、隱藏三種結局）
- **ASCII 動畫**：攻擊、受傷、狂暴、死亡等多種角色狀態圖
- **JSON 存讀檔**：隨時儲存進度，通關後自動清除存檔

## 快速開始

### 方法一：直接執行（已編譯）

```
bin\Debug\net8.0\RPGGame.exe
```

### 方法二：從原始碼建置

```bash
dotnet build
dotnet run
```

> 需要安裝 [.NET 8 SDK](https://dotnet.microsoft.com/download)。

## 遊戲教學

### 主選單

遊戲啟動後顯示主選單，提供以下選項：

```
═══════════════════════════════════
        黑暗勇者傳說
═══════════════════════════════════
  1. 開始新遊戲
  2. 讀取存檔
  3. 離開遊戲
```

- 選 **1** 創建新角色並開始故事
- 選 **2** 載入上次儲存的進度（若存檔不存在則提示）

---

### 角色創建

#### 1. 輸入名字

輸入你的角色名稱（直接按 Enter 使用預設名「勇者」）。

#### 2. 選擇職業

| 職業 | HP 加成 | ATK 加成 | DEF 加成 | MP 加成 | 特色 |
|------|---------|---------|---------|---------|------|
| **戰士** | +30 | +5 | +3 | 0 | 高血量、高防禦，適合初學者 |
| **法師** | 0 | +3 | 0 | +30 | 高 MP，技能傷害更強 |
| **刺客** | +10 | +8 | +1 | +10 | 高攻擊，平衡型玩法 |

> 建議新手選擇**戰士**，職業選擇會影響整場遊戲的打法風格。

---

### 戰鬥系統

戰鬥採回合制，玩家先行。每回合選擇行動：

```
你的回合！選擇行動：
  1. 普通攻擊
  2. 使用技能
  3. 防禦
  4. 查看狀態
```

#### 行動說明

| 行動 | 效果 |
|------|------|
| **普通攻擊** | 造成 ATK × 隨機係數（±12%）的傷害，並累積怒氣 +15 |
| **使用技能** | 消耗 MP，施放選定技能（見技能表） |
| **防禦** | 本回合減傷 50%，進入防禦狀態 |
| **查看狀態** | 顯示角色目前所有數值 |

#### 傷害計算

```
最終傷害 = 基礎攻擊 × 技能倍率 × 隨機係數（0.88 ~ 1.12）− 敵人防禦
```

狂暴狀態下攻擊傷害額外 × 1.5，但有 **20% 機率反噬**（扣自身 HP）。

#### 狀態效果

| 狀態 | 效果 |
|------|------|
| **燃燒** | 每回合扣除約 9 點 HP，持續 3 回合 |
| **暈眩** | 跳過下一回合行動 |
| **防禦中** | 受傷減半，持續至下一次行動 |

### 技能系統

#### 初始技能（1 級可用）

| 技能 | MP 消耗 | 效果 |
|------|---------|------|
| **火焰衝擊** | 20 | 造成高倍物理+魔法傷害，附加燃燒狀態 |
| **治癒術** | 15 | 恢復自身 HP |
| **盾擊** | 10 | 造成傷害並提升自身防禦 |
| **狂暴衝擊** | 25 | 消耗怒氣，造成超高倍傷害 |

#### 進階技能（3 級解鎖）

| 技能 | MP 消耗 | 效果 |
|------|---------|------|
| **冰霜新星** | 30 | 範圍冰系魔法，附加暈眩狀態 |
| **大治癒術** | 25 | 大量回復 HP，效果優於治癒術 |

> **提示**：MP 不足時技能無法施放，請適時使用治癒術保留資源。

### 憤怒 / 狂暴系統

怒氣值是本作的核心機制：

```
怒氣值 ████████░░  80/100
```

- **累積方式**：普通攻擊 +15、受到傷害 +10、防禦時 +5
- **觸發條件**：怒氣達到 100 自動進入**狂暴狀態**
- **狂暴效果**（持續 4 回合）：
  - 所有攻擊傷害 × 1.5
  - 有 20% 機率技能反噬傷害自身
  - 每次狂暴後若累積超過 2 次，**墮落值 +1**

> **注意**：頻繁使用狂暴會提高墮落值，影響結局走向！

---

### 角色成長

擊敗敵人獲得 EXP，EXP 達標後升級：

| 升級效果 | 數值 |
|----------|------|
| 最大 HP | +20 |
| 最大 MP | +10 |
| 攻擊力 | +3 |
| 防禦力 | +2 |

達到 **3 級**後解鎖進階技能。

---

### 存檔與讀檔

- 遊戲在每章關鍵節點後會提示**是否儲存**
- 存檔儲存於 `bin/Debug/net8.0/savegame.json`
- **通關後存檔會自動刪除**，若想重玩請選擇新遊戲

## 劇情與結局

### 故事章節

| 章節 | 地點 | 關鍵事件 |
|------|------|---------|
| **第一章：覺醒** | 村莊廢墟 | 擊退哥布林騎士，面臨是否幫助村民的選擇 |
| **第二章：暗黑森林** | 深林深處 | 對抗暗影幽靈，遭遇接受黑暗力量的誘惑 |
| **第三章：終焉之戰** | 魔王城堡 | 挑戰魔王・夜陌魯斯，決定世界命運 |

### 道德選擇

遊戲中有多個影響結局的選擇：

| 選擇 | 墮落值變化 |
|------|-----------|
| 幫助受困村民 | 不變（正義之路） |
| 無視村民繼續前進 | +1 |
| 接受黑暗力量 | +2，ATK 大幅提升 |
| 在非必要時多次使用狂暴（第 3 次起） | 每次 +1 |

### 三種結局

| 結局 | 解鎖條件 |
|------|---------|
| **光明結局** | 墮落值低，以正義之力擊敗魔王 |
| **黑暗結局** | 接受黑暗力量 **或** 墮落值過高 |
| **隱藏結局** | 在**狂暴狀態**下擊敗最終魔王 |

## 專案架構

```
RPGGame/
├── Program.cs          # 入口點：設定終端機編碼、啟動 GameManager
├── GameManager.cs      # 主遊戲迴圈、章節流程、存讀檔
├── Player.cs           # 玩家屬性、升級、技能、狂暴邏輯
├── Enemy.cs            # 敵人資料與狀態效果
├── BattleSystem.cs     # 戰鬥引擎：回合流程、行動解析
├── SkillSystem.cs      # 技能定義與傷害計算
├── StoryManager.cs     # 劇情文本、分支選擇、結局判定
├── AnimationSystem.cs  # ASCII 動畫渲染、戰鬥畫面更新
├── AsciiArt.cs         # ASCII 圖像資源（角色/敵人各狀態）
├── Utils.cs            # UI 工具：打字動畫、選單、進度條
└── RPGGame.csproj      # 專案設定
```

### 模組相依關係

```
Program
  └── GameManager
        ├── StoryManager   (劇情推進)
        ├── BattleSystem   (戰鬥流程)
        │     ├── Player
        │     ├── Enemy
        │     └── SkillSystem
        ├── AnimationSystem (畫面渲染)
        │     └── AsciiArt
        └── Utils          (共用工具)
```

## 建置方式

### 需求

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

### 指令

```bash
# 建置
dotnet build

# 執行
dotnet run

# 直接執行已編譯的執行檔
./bin/Debug/net8.0/RPGGame.exe   # Windows
./bin/Debug/net8.0/RPGGame       # macOS / Linux
```

### 執行環境注意事項

- 建議使用支援 UTF-8 及 ANSI 色彩的終端機（如 Windows Terminal、iTerm2）
- Windows CMD 可能需要手動設定字體（建議使用 Cascadia Code 或 Consolas）
