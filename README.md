## 故事背景

三年前，魔王**夜陌魯斯**率領魔族大軍入侵，勇者在最後的戰役中身受重傷，從此昏迷不醒。

三年後，你睜開眼睛。

村裡的老祭司告訴你：**魔王的軍隊三天後就會到達這個村子。**

你是他們最後的希望——但你在昏迷的這三年裡，世界已然改變，你的心也在即將到來的旅程中，被迫面臨一次次的抉擇。

**這不只是一場打倒魔王的戰爭，而是一趟關於憤怒、選擇與自我的旅程。**

## 快速開始

### 一鍵啟動（推薦）

**Windows：**
```
雙擊 start.bat
```

**macOS / Linux：**
```bash
chmod +x start.sh
./start.sh
```

### 手動執行

```bash
dotnet run
```

> 需要安裝 [.NET 8 SDK](https://dotnet.microsoft.com/download)。

## 遊戲教學

### 主選單

遊戲啟動後，先選擇**語言**，再進入主選單：

```
  [1] 新遊戲
  [2] 讀取存檔
  [3] 遊玩說明
  [4] 退出遊戲
```

---

### 角色創建

#### 1. 輸入名字

輸入你的角色名稱（直接按 Enter 使用預設名）。

#### 2. 選擇職業

| 職業 | HP | ATK | DEF | MP | 特色 |
|------|-----|-----|-----|-----|------|
| **戰士** | +30 | — | +5 | — | 高血量防禦，怒氣累積快 |
| **法師** | — | +3 | — | +30 | 高 MP，技能傷害強 |
| **刺客** | −10 | +8 | — | — | 極高攻擊，高風險打法 |

> 職業不僅影響數值，也會影響你在故事中的行動方式與最終結局的走向。

---

### 戰鬥系統

戰鬥採回合制，玩家先行。每回合選擇行動：

```
  [1] 普通攻擊
  [2] 使用技能
  [3] 防禦姿態（受到傷害減半）
  [4] 查看狀態
```

#### 傷害計算

```
最終傷害 = 基礎攻擊 × 技能倍率 × 隨機係數（±12%）− 敵人防禦
```

狂暴狀態下攻擊傷害額外 × 1.5，但有 **20% 機率反噬**（扣自身 HP）。

#### 狀態效果

| 狀態 | 效果 |
|------|------|
| **燃燒** | 每回合扣除約 9 點 HP，持續 3 回合 |
| **暈眩** | 跳過下一回合行動 |
| **防禦中** | 受傷減半，持續至下一次行動 |

### 技能系統

#### 初始技能（1 級可用）

| 技能 | 消耗 | 效果 |
|------|------|------|
| **火球術** | MP 15 | 高倍傷害，附加燃燒 |
| **治癒術** | MP 20 | 恢復 35 HP |
| **盾擊** | MP 10 | 傷害＋眩暈機率 |
| **暴走衝擊** | 怒氣 40 | 超高倍傷害＋暴擊機率 |

#### 進階技能（3 級解鎖）

| 技能 | 消耗 | 效果 |
|------|------|------|
| **冰霜新星** | MP 28 | 高倍傷害，必定眩暈 |
| **大治癒術** | MP 32 | 恢復 65 HP |

---

### 憤怒／狂暴系統

怒氣值是本作的核心機制：

```
怒 ████████░░  80/100
```

- **累積方式**：普通攻擊 +15、受到傷害 +12、防禦時 +5
- **觸發條件**：怒氣達到 100 自動進入**狂暴狀態**（4 回合）
- **狂暴效果**：所有攻擊傷害 ×1.5，但 20% 機率反噬自身

> 狂暴狀態強大但並非毫無代價。如何運用憤怒，正是這段旅程的核心課題。

---

### 角色成長

擊敗敵人獲得 EXP，達標後升級：

| 升級效果 | 數值 |
|----------|------|
| 最大 HP | +20 |
| 最大 MP | +10 |
| 攻擊力 | +3 |
| 防禦力 | +2 |

達到 **3 級**後解鎖冰霜新星與大治癒術。

---

### 存檔與讀檔

- 每章結束後**自動儲存**，下次可從斷點繼續
- 存檔路徑：`bin/Debug/net8.0/savegame.json`
- **通關後存檔自動刪除**

## 專案架構

```
RPGGame/
├── start.bat           # Windows 一鍵啟動
├── start.sh            # macOS/Linux 一鍵啟動
├── Program.cs          # 入口點
├── GameManager.cs      # 主遊戲迴圈、章節流程、存讀檔
├── Player.cs           # 玩家屬性、升級、技能、狂暴邏輯
├── Enemy.cs            # 敵人資料與狀態效果
├── BattleSystem.cs     # 戰鬥引擎：回合流程、行動解析
├── SkillSystem.cs      # 技能定義與傷害計算
├── StoryManager.cs     # 劇情文本、分支選擇、結局判定
├── Localization.cs     # 多語系字串（ZH / EN / JA）
├── AnimationSystem.cs  # ASCII 動畫渲染
├── AsciiArt.cs         # ASCII 圖像資源
└── Utils.cs            # UI 工具：打字動畫、選單、進度條
```

## 建置方式

### 需求

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

### 指令

```bash
dotnet build   # 建置
dotnet run     # 執行
```

### 終端機建議

- **Windows**：Windows Terminal（支援 UTF-8 與 ANSI 色彩）
- **macOS**：iTerm2 或原生 Terminal
- **Linux**：任何支援 UTF-8 的終端機
