# Skill: Console RPG Generator（純程式 RPG 產生器）

## 目標

建立一個「可執行的 Console RPG 遊戲」，使用純 C# 程式碼，不依賴遊戲引擎。

---

## 技術要求

* 語言：C#
* 執行環境：.NET 8 Console Application
* 可在 Visual Studio 2022 / dotnet CLI 直接執行

---

## 專案結構

```
RPGGame/
├── Program.cs          # 入口點（支援 --test 參數）
├── GameManager.cs      # 主遊戲迴圈、章節流程、存讀檔
├── Player.cs           # 玩家屬性、升級、技能、狂暴邏輯
├── Enemy.cs            # 敵人資料（13 種）
├── BattleSystem.cs     # 戰鬥引擎
├── SkillSystem.cs      # 職業專屬技能
├── StoryManager.cs     # 劇情文本、幽默支線、9 種結局
├── GameTests.cs        # 自動化測試套件
├── Localization.cs     # 三語支援（ZH / EN / JA）
├── AnimationSystem.cs  # ASCII 動畫
├── AsciiArt.cs         # ASCII 圖像
└── Utils.cs            # UI 工具函式庫
```

## 核心系統

### 1. 玩家系統

* HP / MP / EXP / Level
* Attack / Defense
* 5 種職業（各有專屬技能、故事台詞與隱藏結局）
* 道德選擇：腐化值影響結局分支

---

### 2. 戰鬥系統

* 回合制戰鬥（玩家先行）
* 行動選單：普通攻擊 / 使用技能 / 防禦姿態 / 查看狀態
* 傷害計算：`ATK × 倍率 × ±12% 隨機 − 敵防禦`
* 狀態效果：燃燒、中毒、眩暈、防禦中

---

### 3. 技能系統

* 每職業 4 個基礎技能 + 2 個 3 級進階技能
* 消耗 MP 或怒氣值
* 特殊效果：暴擊、燃燒、眩暈、吸血、範圍攻擊

---

### 4. 劇情系統

* 三章完整故事
* 每章有道德選擇影響結局
* 3 條幽默支線劇情（龍王轉世史萊姆、迷路外送魔族、魔王城員工休息室）
* 玩家對話與敘事依職業動態變化

---

### 5. 特殊系統

**暴走系統**

* 攻擊 / 受傷 / 防禦時累積怒氣
* 滿 100 自動進入狂暴狀態（4 回合）
* 狂暴效果：ATK × 1.5，但有 20% 機率反噬
* 部分結局與隱藏結局需以狂暴狀態擊敗最終 Boss

---

## 遊戲流程

1. 語言選擇（ZH / EN / JA）
2. 角色創建（命名 + 職業）
3. 第一章：覺醒
   - 教學戰鬥（史萊姆）
   - 支線：龍王轉世史萊姆
   - 隨機漫遊遭遇（50% 觸發）
   - 道德選擇：是否救老婆婆
4. 第二章：暗黑森林
   - 隨機 Tier2A 遭遇戰
   - 道德選擇：是否吸收黑暗符石
   - 支線：迷路的魔族外送員
   - 隨機 Tier2B 遭遇戰
5. 第三章：魔王城
   - 前哨戰：黑暗聖騎士
   - 支線：魔王城員工休息室
   - 最終選擇：是否接受魔王的交易
   - 最終戰：魔王・夜陌魯斯
   - 結局判定（9 種）

---

## 結局系統

共 9 種結局，由以下條件組合決定：

| 結局 | 條件 |
|------|------|
| ✦ 純淨之道（隱藏） | 零腐化、未狂暴、幫助村民、拒絕黑暗 |
| ★ 暴走聖者（隱藏） | 狂暴中打倒最終 Boss，腐化值 ≤ 1 |
| ✦ 聖光的歸途 | 聖騎士限定 |
| ✦ 荒野的呼喚 | 遊俠限定 |
| 光明的彼岸 | 協助村民、低腐化 |
| 孤獨的勝者 | 未幫助村民 |
| 業火之中的行者 | 接觸黑暗但未墮落 |
| 黑暗的繼承者 | 腐化值 ≥ 4 |
| 魔王轉世 | 腐化值 ≥ 6 |

## 程式測試

```bash
# Windows
雙擊 test.bat

# macOS / Linux
./test.sh

# 手動
dotnet run -- --test
```

134 項自動化測試覆蓋敵人工廠、隨機遭遇池、玩家創建、戰鬥邏輯與狀態效果邊界。

## 限制

* 不可只輸出文字劇情 — 必須包含完整可執行程式
* 必須有戰鬥系統（已實作）
* 必須可在 Console 執行（已實作）
* 必須可互動（玩家輸入）（已實作）

---

## 輸出規格

1. 完整 C# 程式碼
2. 可直接 `dotnet run` 執行
3. 不需額外套件
